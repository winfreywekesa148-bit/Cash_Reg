# CASH_REG
 
 ## Description
 The Cash_Reg accepts the item, price and quantity. It adds the total price and calculates the final amount after removing the discount. It also stores the data of the item purchased. This system is mostly used in places like mini-marts and the likes, which has reduced the number made by humans.

## Usage
